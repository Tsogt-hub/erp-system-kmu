# Hero ERP Angebotserstellung - Analyse

## Datum: 03.11.2025

### Funktionen die benötigt werden:
1. **Angebotspositionen**:
   - Artikel/Produkt auswählen
   - Menge
   - Einzelpreis
   - Rabatt (%)
   - Gesamtpreis pro Position
   - Steuersatz pro Position

2. **Angebotsdetails**:
   - Angebotsnummer (automatisch generiert)
   - Kunde
   - Projekt (optional)
   - Gültigkeitsdatum
   - Status (Entwurf, Versendet, Angenommen, Abgelehnt)
   - Notizen

3. **Preisberechnung**:
   - Zwischensumme (netto)
   - Steuern
   - Gesamtsumme (brutto)
   - Rabatte

4. **PDF-Export**:
   - Professionelles Angebotsdokument
   - Firmenlogo
   - Alle Positionen
   - Gesamtbeträge
